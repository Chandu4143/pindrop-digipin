## What is DIGIPIN?

DIGIPIN (Digital Postal Index Number) is India Post's system for precise location identification. It converts any GPS location into a simple 10-character code like `3PJ-7KL-4MN2`.

Think of it as a "what3words" for India, but official and standardized.
